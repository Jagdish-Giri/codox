# Codox - Learn, Build, Grow.

Hey there! Welcome to codox - a platform I've been working on to make learning web development more engaging and fun.

## What's This All About?

You know how most learning platforms are just boring video lectures? Yeah, I wanted to change that. Codox brings gamification into the mix - think XP points, levels, achievements, and even a leaderboard. It's like turning your coding journey into a game!

I built this because I noticed my friends losing interest in traditional courses. So I thought, "Why not make it more interactive?" And here we are.

## Cool Features I've Added
```
-Gamified Learning : Earn XP, unlock achievements, climb the leaderboard. Makes studying actually fun!
-Multiple Paths :  Whether you're into frontend, backend, or full-stack - there's something for everyone
-AI Chat Assistant : Stuck? Just ask! (Though I'm still working on making it smarter) 
-Clean Design : No clutter, just clean green and black theme that's easy on the eyes
-Mobile Friendly : easy and mobile friendly easy to use for mobile which is good for mobile learner's
```

## Tech Stack (What I Used)

Started simple and kept it that way:

Frontend :

- React (because honestly, what else?)
- React Router for navigation
- React Icons for those nice little icons
- Plain CSS (yeah, no Tailwind or Bootstrap - wanted full control)

Backend (Coming Soon) :

Planning to add Node.js + MongoDB soon. Need to figure out real-time features first.
